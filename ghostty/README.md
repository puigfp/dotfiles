# Ghostty Configuration

## Keybindings

### Tab Navigation
| Keybinding | Action |
|------------|--------|
| `cmd+h` | Switch to previous tab |
| `cmd+l` | Switch to next tab |
| `cmd+,` | Rename current tab |

### Tab Reordering
| Keybinding | Action |
|------------|--------|
| `cmd+shift+h` | Move tab left |
| `cmd+shift+l` | Move tab right |

### Split Creation
| Keybinding | Action |
|------------|--------|
| `ctrl+\` | Vertical split (right) |
| `ctrl+shift+\` | Horizontal split (down) |

### Split Navigation (vim-style)
| Keybinding | Action |
|------------|--------|
| `ctrl+h` | Focus split left |
| `ctrl+j` | Focus split down |
| `ctrl+k` | Focus split up |
| `ctrl+l` | Focus split right |

### Split Resizing (vim-style)
| Keybinding | Action |
|------------|--------|
| `ctrl+shift+h` | Resize split left |
| `ctrl+shift+j` | Resize split down |
| `ctrl+shift+k` | Resize split up |
| `ctrl+shift+l` | Resize split right |
